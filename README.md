# Planner
1. Приложение создано для изучения .NET MAUI Blazor(MudBlazor).
2. Поможет рассчитать цифры необходимые для выполнения плановых показателей.

   Ссылка на RuStore: https://apps.rustore.ru/app/com.artsoft.planner
